Landing page de alta costura para la marca **Hidalguería**, estudio de diseño interior de lujo en Ciudad de México.

**Stack:** React 19 + TypeScript + Vite 8 + Framer Motion + Tailwind CSS v4

**Por qué:** El cliente quería una presencia web que comunicara nivel de revista editorial (Architectural Digest, AD México) sin navbar, solo scroll cinematico.

**How to apply:** Al modificar el proyecto, preservar la paleta (cream/sand/dune/ash/graphite/obsidian/gold definidas en index.css), las tipografías (Cormorant Garamond + Raleway 300) y el estilo de animación (ease `[0.16, 1, 0.3, 1]`). El contacto es únicamente Instagram y WhatsApp — sin formularios.
